# Misc. Receiving Template

## Purpose

A simpler electronic receiving template for workflows that still rely on manual quantity entry rather than scan-based validation.

## Features

- Area to input a packing list
- Conditional formatting similar to the Company C workflow
  - Green when difference equals zero
  - Yellow when quantity received exceeds expected quantity
- Automatic difference calculation
- Works on phones and scanners

## Current Workflow

Paper-and-pencil receiving with bin locations and notes written directly on the packing slip.

## Updated Workflow

The new workflow keeps the same basic receiving process but moves it into an electronic format.

## Benefits

- Reduces paper usage
- Makes receiving information easier to read and maintain
- Adds simple visual validation through conditional formatting
- Makes the process more accessible on mobile devices and scanners

## Notes

This template is a flexible, lower-complexity option for receiving workflows that do not need full scan-column logic.
