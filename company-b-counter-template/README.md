# Company B Counter Template

## Stack

Google Sheets + Apps Script automation

## Problem

The current workflow for Company B products often involves receiving units, writing serial numbers by hand, counting on paper, and then manually entering the results into Company B's database. This process is slow and introduces multiple opportunities for error.

## Purpose

This template provides a scanner-based counting workflow for products with serial numbers, with built-in duplicate detection and automatic counting.

## How It Helps

- Eliminates manual handwritten serial number entry
- Reduces duplicate entry mistakes
- Removes the need for separate manual counting
- Makes scanner-based workflows practical on mobile devices and scanners
- Makes it easy to copy and paste results into Company B sheets

## Features

- Independent columns that each act as their own category
- Automatic counters within each column
- Duplicate detection logic
- Flexible layout that can use as many or as few columns as needed
- Printable format with ready-made categories

## Use Cases

### Picking and Receiving Company B Units

- Create categories for different unit types
- Scan serial numbers into the matching columns
- Let the sheet count automatically
- Avoid recounting and duplicate checking
- Copy and paste results into Company B documentation

### General Serial Number Workflows

Anything with serial numbers can potentially use this template.

### Mobile Scanner Workflow Example

A worker can walk a shipment, scan units directly into the sheet from the scanner, and complete the data collection process without having to stop and manually write everything down first.

## Reported Benefits

- Efficiency increase of roughly 300 percent in some workflows
- Tasks that took 30 or more minutes can drop to around 10 minutes
- Significant accuracy improvement when using barcode scans instead of handwritten entry
- Easier identification of missing units
- Reduced money lost from picking errors
- Easier for more people to perform accurately
- Greater independence from Company B's system

## Real Run Example

- POs: 124144-00, 124126-00, 858, 859
- 56 units
- 7 different models
- Quantities checked, serial numbers recorded exactly, and entered into Company B's system in 19 minutes and 15 seconds

## Expansion Potential

Any workflow based around serial numbers is a candidate for this tool.
